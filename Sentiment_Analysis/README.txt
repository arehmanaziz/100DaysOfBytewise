Sentiment Analysis Application
==============================
(Webpage: https://sentiment-analysis-bytewise.streamlit.app)

Overview:
---------
This project is a sentiment analysis application that predicts the sentiment (positive, neutral, or negative) of a given text input using a machine learning model. The model has been trained and optimized to provide accurate sentiment predictions.

Setup Instructions:
-------------------
1. Install Dependencies:
Before running the application, make sure you have all the required Python packages installed. The key dependency for this project is streamlit. You can install it using pip:

'pip install streamlit'

2. Run the Application:
Once you have installed the necessary dependencies, you can run the Streamlit application using the following command:

'streamlit run streamlit_app.py'

3. Important Note:
Make sure to open your terminal in the folder where App.py is located. If your terminal is not in the correct folder, you will encounter a "file not found" error.
